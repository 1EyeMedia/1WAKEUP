# Focus Timer Tool

## Introduction
The Focus Timer Tool is designed to help users enhance their productivity by allowing them to manage their time efficiently. The tool employs the Pomodoro Technique, which involves working in focused bursts followed by short breaks.

## Features
- **Customizable Timer**: Users can set their preferred durations for work sessions and breaks.
- **Notifications**: Alerts notify users when to start and stop work sessions and breaks.
- **Statistics Tracking**: Keep track of completed sessions and breaks.
- **User-Friendly Interface**: Simple and intuitive design for ease of use.

## Installation
To install the Focus Timer Tool, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/1EyeMedia/1WAKEUP.git
   ```
2. Navigate into the project directory:
   ```bash
   cd 1WAKEUP
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
1. Launch the application:
   ```bash
   npm start
   ```
2. Set your preferred work session and break durations in the settings menu.
3. Start a session by clicking the "Start" button.
4. Focus on your task until the timer alerts you for a break.
5. Enjoy your break before starting the next session.
